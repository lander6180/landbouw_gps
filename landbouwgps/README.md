# Landbouw GPS – Werkbreedte & Heading

Een eenvoudige GPS-webapp voor landbouwmachines:

- Live GPS-positie
- Automatische heading (richting) op basis van eerste meters
- Instelbare werkbreedte (3 m, 6 m, 12 m, …)
- Parallelle lijnen op kaart (Leaflet)
- Werkt op smartphone via HTTPS (GitHub Pages)

## Gebruik

1. Open de website op je smartphone.
2. Rij 5–10 meter vooruit zodat de app je richting kan bepalen.
3. Druk op **Start eerste rij**.
4. De kaart toont nu alle banen op jouw werkbreedte.
